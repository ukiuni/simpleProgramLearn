# simpleProgramLearn
りんごをたすあぷり

ChromeなどデベロッパーツールのコンソールでJavaScriptを覚えます。

つかえる機能は以下。

## もってくる(すうじ)
## ふやす(すうじ)
りんごをふやします。たしざんです。

## たべる(すうじ)
## ひく(すうじ)
りんごをへらします。ひきざんです。

## かける(すうじ)
りんごをにんずうぶんふやします。かけざんです。

## わける
りんごをにんずうぶんにわけます。わりざんです。

## もし(すうじ, じょうけん, やること)
「じょうけん」がそろえば、「やること」をします。
もし(14, おおい(5), ひく(3), だす)
のようにつかえます。
「じょうけん」でつかえるのは
### おおい(ひかくするすうじ)
### すくない(ひかくするすうじ)
です。「じょうけん」は「すうじ」と「ひかくするすうじ」をくらべます。
